# Duty Hours App

A comprehensive web application for posting duties and tracking duty hours with admin verification and user management.

## Features

- **Admin Dashboard**: Post duties, verify signups, monitor progress, mark duties as completed
- **User Dashboard**: Browse available duties, sign up, track personal duty status
- **Duty Tracking**: Automatic hour accumulation and long-term record keeping
- **Mobile Responsive**: Works on all devices, can be installed as PWA
- **Real-time Updates**: Powered by Supabase backend

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Supabase (Authentication & Database)
- Responsive Design
- Progressive Web App (PWA)

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Sign in with your account
4. Start managing duties!

## Deployment

Deployed on Netlify for free hosting.
